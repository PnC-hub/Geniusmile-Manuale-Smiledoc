# Primo Giorno

## Obiettivo

Accogliere il nuovo collaboratore e fornirgli tutto il necessario per sentirsi parte del team fin dal primo momento.

## Checklist Primo Giorno

### Prima dell'Arrivo

- [ ] Preparare postazione di lavoro
- [ ] Creare credenziali software gestionale
- [ ] Preparare badge/chiavi
- [ ] Informare il team dell'arrivo
- [ ] Preparare kit di benvenuto

### Kit di Benvenuto

| Divisa | Con logo studio |
| Badge identificativo | Nome e ruolo |

### Regole Base

1. **Puntualità**: arrivare 10 minuti prima dell'orario
2. **Divisa**: sempre pulita e ordinata
3. **Cellulare**: in modalità silenziosa, uso personale solo in pausa
4. **Riservatezza**: mai parlare di pazienti fuori dallo studio
