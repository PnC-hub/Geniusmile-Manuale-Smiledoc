# Comunicazione con il Paziente

## Principi di Comunicazione

### Valori Guida

| Valore | Significato |
|--------|-------------|
| **Chiarezza** | Linguaggio comprensibile, no gergo tecnico |
| **Empatia** | Ascolto attivo, comprensione delle emozioni |
| **Trasparenza** | Informazioni complete su trattamenti e costi |
| **Rispetto** | Dignità del paziente, privacy |
| **Professionalità** | Competenza e cortesia sempre |

## Standard di Comunicazione

### Accoglienza

| Elemento | Standard |
|----------|----------|
| Saluto | Sempre con nome del paziente |
| Sorriso | Sempre |
| Contatto visivo | Diretto e cordiale |
| Postura | Aperta e accogliente |
| Tono | Caldo e rassicurante |

### Al Telefono

| Elemento | Standard |
|----------|----------|
| Risposta | Entro 3 squilli |
| Presentazione | Nome studio e proprio nome |
| Tono | Sorridente (si percepisce) |
| Ascolto | Completo prima di rispondere |
| Chiusura | Ringraziamento e saluto cordiale |

### In Studio

| Momento | Comunicazione |
|---------|---------------|
| Arrivo | Accoglienza, verifica anagrafica |
| Attesa | Informazioni su tempistiche |
| Trattamento | Spiegazione passaggi principali |
| Congedo | Istruzioni, prossimo appuntamento |

## Comunicazione Clinica

### Prima del Trattamento

- [ ] Spiegare cosa verrà fatto
- [ ] Illustrare eventuali alternative
- [ ] Informare su rischi e benefici
- [ ] Verificare comprensione
- [ ] Raccogliere consenso

### Durante il Trattamento

- [ ] Avvisare prima di procedure sensibili
- [ ] Chiedere periodicamente se va tutto bene
- [ ] Informare sui progressi
- [ ] Rispondere a eventuali segnali di disagio

### Dopo il Trattamento

- [ ] Spiegare cosa è stato fatto
- [ ] Fornire istruzioni post-operatorie (anche scritte)
- [ ] Indicare quando chiamare in caso di problemi
- [ ] Programmare eventuale controllo

## Gestione Situazioni Difficili

### Paziente Ansioso

| Azione | Come |
|--------|------|
| Riconoscere l'ansia | "Capisco che possa essere preoccupato" |
| Rassicurare | Spiegare ogni passaggio |
| Dare controllo | "Alzi la mano se vuole fermarsi" |
| Procedere gradualmente | Pause se necessario |

### Paziente Insoddisfatto

| Fase | Azione |
|------|--------|
| Ascolto | Lasciare sfogare senza interrompere |
| Empatia | "Capisco il suo disagio" |
| Chiarimento | Comprendere il problema specifico |
| Soluzione | Proporre rimedio concreto |
| Follow-up | Verificare risoluzione |

### Comunicazione di Cattive Notizie

1. **Preparare** l'ambiente (privacy, tempo)
2. **Avvisare** che c'è qualcosa di importante
3. **Comunicare** in modo chiaro e diretto
4. **Permettere** reazioni emotive
5. **Rispondere** alle domande
6. **Proporre** percorso e supporto

## Comunicazione Scritta

### Email e Messaggi

| Elemento | Standard |
|----------|----------|
| Risposta | Entro 24 ore lavorative |
| Tono | Professionale ma cordiale |
| Contenuto | Chiaro e completo |
| Firma | Nome, ruolo, contatti studio |

### Documenti al Paziente

| Documento | Caratteristiche |
|-----------|-----------------|
| Preventivo | Dettagliato, comprensibile |
| Istruzioni | Chiare, con grafica se utile |
| Informative | Linguaggio accessibile |
| Ricevute | Complete e conformi |

## Comunicazione Non Verbale

### Elementi Chiave

| Elemento | Positivo | Da Evitare |
|----------|----------|------------|
| Postura | Aperta, leggermente inclinata | Braccia incrociate |
| Sguardo | Contatto visivo regolare | Fissare o evitare |
| Espressione | Sorriso, interesse | Espressione annoiata |
| Distanza | Rispettosa, non invadente | Troppo vicino/lontano |
| Gesti | Calmi, rassicuranti | Nervosi, bruschi |

## Formazione alla Comunicazione

### Competenze da Sviluppare

| Competenza | Modalità |
|------------|----------|
| Ascolto attivo | Role playing, feedback |
| Gestione emozioni | Training, supervisione |
| Comunicazione efficace | Corso specifico |
| Gestione conflitti | Workshop |

### Valutazione

- Feedback pazienti
- Osservazione diretta
- Mystery patient (se applicabile)
- Autovalutazione
