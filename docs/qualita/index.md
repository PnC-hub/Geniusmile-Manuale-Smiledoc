## Politica della Qualità

Lo Studio si impegna a:

1. **Eccellenza clinica**: trattamenti secondo le migliori evidenze scientifiche
2. **Centralità del paziente**: ogni decisione parte dalle esigenze del paziente
3. **Sicurezza**: ambiente sicuro per pazienti e operatori
4. **Miglioramento continuo**: analisi e ottimizzazione costante
5. **Formazione**: personale competente e aggiornato

## Obiettivi di Qualità

| Area | Obiettivo | Target |
|------|-----------|--------|
| Soddisfazione | NPS pazienti | ≥ 50 |
| Puntualità | Rispetto orari appuntamenti | ≥ 90% |
| Sicurezza | Infezioni post-operatorie | 0% |
| Efficienza | Completamento piani cura | ≥ 80% |
| Fidelizzazione | Ritorno pazienti | ≥ 70% |
