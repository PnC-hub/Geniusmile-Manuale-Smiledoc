# Sistema HR {{ clinic.name }}

Il **Sistema HR {{ clinic.name }}** è una piattaforma integrata per la gestione delle risorse umane, progettata per digitalizzare e automatizzare i processi HR dello studio odontoiatrico.

## Funzionalità Principali

### Assistente AI

L'assistente virtuale HR utilizza intelligenza artificiale per rispondere alle domande dei dipendenti su:

- Procedure operative
- Normative di sicurezza
- Protocolli clinici
- Documentazione aziendale

L'assistente è disponibile 24/7 e può escalare le richieste complesse al team HR.

### Formazione

Sistema completo per la gestione della formazione obbligatoria e volontaria:

- **Percorsi formativi** strutturati per ruolo
- **Quiz di verifica** con punteggio minimo
- **Certificati digitali** con QR code verificabile
- **Reminder automatici** per scadenze

### Documenti e Presa Visione

Gestione digitale della documentazione con:

- **Firma digitale** (click o FEA)
- **Hash SHA-256** per integrità
- **Tracciabilità completa** (IP, timestamp)
- **Verifica pubblica** tramite hash

### Proposte (Wiki Controllata)

Sistema per raccogliere e gestire proposte di miglioramento:

- Invio proposte da parte dei dipendenti
- Workflow di approvazione
- Storico modifiche
- Integrazione con knowledge base

### Notifiche

Sistema multi-canale per comunicazioni HR:

- Notifiche in-app
- Email automatiche
- Reminder scadenze
- Avvisi urgenti

## Accesso al Sistema

### Per i Dipendenti

1. Accedi al CRM con le tue credenziali
2. Vai alla sezione "HR" dal menu principale
3. Completa l'onboarding se sei un nuovo dipendente

### Per gli Amministratori

La dashboard admin è accessibile ai ruoli:

- Coordinatore
- Admin

Funzionalità admin:

- Gestione escalation AI
- Statistiche formazione
- Invio notifiche massive
- Monitoraggio compliance

## Widget Chat

Il widget chat può essere integrato in qualsiasi pagina web:

```html
<script src="https://api.geniusmile.com/hr/widget/hr-chat-widget.js"
        data-api-key="YOUR_API_KEY"
        data-dipendente-id="123">
</script>
```
