### STERILIZZAZIONE

- [ ] Controllo scorte prodotti critici:

### ATTREZZATURE

- [ ] Riuniti: funzionamento verificato
- [ ] Compressore: livello olio, drenaggio condensa
- [ ] Aspirazione: filtri verificati
- [ ] Ultrasuoni: funzionamento
- [ ] RX: funzionamento

### PULIZIE

- [ ] Pulizia profonda eseguita
- [ ] Aree critiche verificate
- [ ] Prodotti pulizia sufficienti

### AMMINISTRAZIONE

- [ ] Fatture emesse controllate
- [ ] Crediti scaduti verificati
- [ ] Backup dati eseguito

### COMUNICAZIONE

- [ ] Recall della settimana inviati
- [ ] Conferme appuntamenti settimana prossima
- [ ] Recensioni verificate e gestite
