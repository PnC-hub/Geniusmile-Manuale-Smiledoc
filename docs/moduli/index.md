## Utilizzo dei Moduli

### Principi Generali

1. **Utilizzare sempre** i moduli standard approvati
2. **Non modificare** i moduli senza autorizzazione
3. **Compilare** in modo leggibile e completo
4. **Conservare** secondo le indicazioni di archiviazione
5. **Aggiornare** quando richiesto

### Gestione Versioni

Ogni modulo riporta:

- Codice identificativo
- Versione
- Data ultima revisione
- Responsabile approvazione

## Responsabilità

| Ruolo | Compito |
|-------|---------|
| Segreteria | Distribuzione, raccolta, archiviazione |
| Personale clinico | Compilazione corretta |
| Coordinatore | Verifica completezza |
| Direzione | Approvazione nuove versioni |
