# Ortopantomografia (OPT)

## Obiettivo
Eseguire ortopantomografie di qualità diagnostica ottimale con la minima dose al paziente.

## Indicazioni

### Indicazioni Appropriate

| Situazione | Utilità |
|------------|---------|
| Prima visita (adulti) | Valutazione generale |
| Valutazione ortodontica | Sviluppo dentale, simmetria |
| Chirurgia estrattiva (terzi molari) | Rapporti anatomici |
| Implantoprotesi (screening) | Valutazione iniziale |
| Patologia ossea | Lesioni, cisti |
| Trauma maxillo-facciale | Fratture |

### Indicazioni NON Appropriate

- Ricerca carie interprossimali (preferire bitewing)
- Valutazione endodontica di precisione (preferire periapicali)
- Follow-up parodontale di routine

## Apparecchiatura

### Componenti Principali

- Tubo radiogeno rotante
- Sensore/cassetta rotante
- Sistema di posizionamento (mentoniera, appoggi)
- Pannello comandi

### Controlli Pre-Utilizzo

- [ ] Apparecchio acceso e in standby
- [ ] Sensore/cassetta in posizione
- [ ] Posizionatori puliti
- [ ] Parametri impostati

## Procedura di Esecuzione

### Preparazione Paziente

1. **Rimuovere**:
   - Occhiali
   - Orecchini
   - Collane
   - Apparecchi acustici
   - Protesi mobili (secondo indicazione)
   - Piercing (se possibile)

2. **Informare** il paziente:
   - Durata (15-20 secondi)
   - Necessità di stare immobili
   - Cosa sentirà (rotazione)

3. **Protezione**:
   - Collare piombato per la tiroide

### Posizionamento

!!! warning "Posizionamento Critico"
    Un corretto posizionamento è essenziale per la qualità dell'immagine.

```
PUNTI DI RIFERIMENTO

1. PIANO DI FRANCOFORTE orizzontale
   (margine inferiore orbita - meato acustico esterno)

2. PIANO SAGITTALE MEDIANO centrato
   (simmetria dx/sx)

3. INCISIVI nel solco della mentoniera
   (focus layer)

4. LINGUA contro il palato
   (evita ombra radiopaca)

5. LABBRA chiuse
```

### Sequenza Operativa

1. Posizionare il paziente in piedi (o seduto)
2. Regolare l'altezza dell'apparecchio
3. Far mordere la mentoniera con gli incisivi
4. Posizionare gli appoggi temporali
5. Verificare piano di Francoforte
6. Chiedere di deglutire e appoggiare la lingua al palato
7. Verificare immobilità
8. Allontanarsi e attivare l'esposizione
9. Attendere fine rotazione

### Parametri di Esposizione

| Parametro | Range Tipico |
|-----------|--------------|
| kV | 66-74 |
| mA | 8-12 |
| Tempo | 12-18 secondi |

I parametri vanno adattati in base a:

- Corporatura del paziente
- Indicazioni del produttore
- Protocollo validato dall'Esperto di Radioprotezione

## Controllo Qualità

### Criteri di Valutazione

| Parametro | Accettabile | Non Accettabile |
|-----------|-------------|-----------------|
| Simmetria | Condili alla stessa altezza | Asimmetria evidente |
| Francoforte | Orizzontale | Inclinato |
| Focus | Denti a fuoco | Sfocatura anteriore |
| Colonna | Non sovrapposta | Sovrapposizione marcata |
| Densità | Uniforme | Troppo chiara/scura |

### Errori Comuni di Posizionamento

| Errore | Risultato | Correzione |
|--------|-----------|------------|
| Paziente troppo avanti | Denti anteriori stretti | Arretrare |
| Paziente troppo indietro | Denti anteriori larghi, sfocati | Avanzare |
| Mento alto | Appiattimento piano occlusale | Abbassare mento |
| Mento basso | Curva del sorriso accentuata | Alzare mento |
| Testa ruotata | Asimmetria, ramo ingrandito | Centrare |
| Lingua bassa | Ombra radiopaca sul palato | Lingua al palato |

## Casi Particolari

### Pazienti Pediatrici

- Utilizzare programma pediatrico (se disponibile)
- Ridurre parametri di esposizione
- Attenzione al posizionamento (altezza)
- Rassicurare durante l'esame

### Pazienti con Disabilità

- Valutare la capacità di stare immobili
- Considerare sedazione se necessario
- Eventuale accompagnatore (con protezione)

### Pazienti Edentuli

- Rimuovere protesi (o lasciare secondo indicazione)
- Posizionamento con roll di cotone se necessario

## Documentazione

### Registrazione

Per ogni esame registrare:

| Campo | Informazione |
|-------|--------------|
| Data | Esame |
| Paziente | Dati identificativi |
| Indicazione | Motivazione clinica |
| Parametri | kV, mA, tempo |
| Operatore | Nome e firma |
| Qualità | Valutazione immagine |

### Refertazione

L'odontoiatra deve refertare descrivendo:

- Strutture ossee (mascellare, mandibola, ATM)
- Strutture dentali
- Eventuali reperti patologici
- Conclusioni diagnostiche

## Archiviazione

- Formato DICOM (digitale)
- Backup periodico
- Conservazione minimo 10 anni
- Accessibilità per paziente/altri professionisti

## Responsabilità

| Ruolo | Compito |
|-------|---------|
| Odontoiatra | Indicazione, supervisione, refertazione |
| Personale formato | Posizionamento, esecuzione |
| Coordinatore | Manutenzione, registri |
