# Radiologia Odontoiatrica

L'utilizzo di radiazioni ionizzanti in odontoiatria richiede il rispetto di protocolli rigorosi per garantire la sicurezza del paziente e degli operatori.

## Principi Fondamentali

### Principio ALARA

**A**s **L**ow **A**s **R**easonably **A**chievable

Ogni esposizione deve essere:

- **Giustificata**: beneficio > rischio
- **Ottimizzata**: dose minima necessaria
- **Limitata**: nel rispetto dei limiti di legge

### Giustificazione dell'Esame

Prima di ogni esame radiografico valutare:

1. L'esame è necessario?
2. È già disponibile un esame recente?
3. Esistono alternative non radianti?
4. Il beneficio diagnostico supera il rischio?

## Personale Autorizzato

### Requisiti

| Ruolo | Abilitazione |
|-------|--------------|
| Odontoiatra | Laurea + formazione radioprotezione |
| Igienista | NO (può posizionare, non esporre) |
| Assistente | NO (può posizionare, non esporre) |

### Formazione

- Corso radioprotezione iniziale
- Aggiornamento periodico (ogni 5 anni)
- Registro formazione aggiornato
