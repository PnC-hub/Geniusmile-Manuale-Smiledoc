# Reazioni Allergiche

## Obiettivo
Riconoscere e trattare tempestivamente le reazioni allergiche, prevenendo l'evoluzione verso forme gravi.

## Classificazione

### Reazioni Lievi (Tipo I - Localizzate)

**Sintomi:**

- Prurito localizzato
- Eritema cutaneo limitato
- Orticaria circoscritta
- Edema locale

**Gravità:** Bassa - Gestibile in studio

### Reazioni Moderate (Tipo II)

**Sintomi:**

- Orticaria diffusa
- Angioedema (labbra, palpebre)
- Nausea, vomito
- Crampi addominali
- Prurito generalizzato

**Gravità:** Media - Richiede attenzione, possibile 118

### Reazioni Gravi - Anafilassi (Tipo III)

!!! danger "Emergenza Critica"
    L'anafilassi può essere fatale. Agire immediatamente.

**Sintomi:**

- Difficoltà respiratoria (broncospasmo)
- Edema della glottide (voce rauca, stridore)
- Ipotensione grave
- Tachicardia
- Perdita di coscienza
- Cianosi

**Gravità:** Critica - 118 immediato

## Allergeni Comuni in Odontoiatria

| Categoria | Allergeni | Alternativa |
|-----------|-----------|-------------|
| Anestetici | Articaina, Lidocaina | Mepivacaina, test allergologico |
| Antibiotici | Penicilline | Macrolidi, Clindamicina |
| FANS | Aspirina, Ibuprofene | Paracetamolo |
| Lattice | Guanti, diga | Guanti nitrile |
| Metalli | Nichel | Materiali nichel-free |
| Antisettici | Iodopovidone | Clorexidina |

## Prevenzione

### Anamnesi Allergologica

Domande specifiche:

1. "Ha mai avuto reazioni allergiche a farmaci?"
2. "Ha mai avuto problemi con anestetici dal dentista?"
3. "È allergico al lattice o alla gomma?"
4. "Ha allergie alimentari note?"
5. "Ha mai avuto reazioni con metalli (gioielli)?"

### Pazienti a Rischio

Identificare e segnalare in cartella:

- Atopici (asma, eczema, rinite allergica)
- Precedenti reazioni allergiche
- Allergie multiple
- Allergie gravi note (anafilassi)

### Precauzioni

Per pazienti allergici noti:

- Evidenziare in cartella (**ALERT**)
- Verificare prima di ogni seduta
- Evitare allergeni noti
- Kit emergenza verificato e a portata
- Informare tutto il team

## Protocollo di Trattamento

### Reazione Lieve

```
1. INTERROMPERE trattamento
2. RIMUOVERE eventuale causa (guanti lattice, farmaco)
3. RASSICURARE il paziente
4. MONITORARE per 30-60 minuti
5. ANTISTAMINICO per bocca (se disponibile)
6. VALUTARE necessità di approfondimento allergologico
```

### Reazione Moderata

```
1. INTERROMPERE trattamento
2. RIMUOVERE causa scatenante
3. POSIZIONE comoda (semiseduta)
4. CHIAMARE AIUTO
5. ANTISTAMINICO per bocca o IM
6. CORTISONICO se disponibile
7. MONITORARE attentamente
8. CHIAMARE 118 se peggiora
```

### Anafilassi

!!! danger "Azione Immediata"
    Non attendere l'evoluzione. Ogni secondo conta.

```

├─────────────────────────────────────────┤
│ 1. CHIAMARE IL 118 IMMEDIATAMENTE       │
│                                         │
│ 2. POSIZIONE:                           │
│    - Se cosciente: supina, gambe alzate │
│    - Se dispnea: semiseduta             │
│    - Se incosciente: PLS               │
│                                         │
│ 3. ADRENALINA IM (se disponibile):      │
│    - Adulto: 0.5 mg (1:1000)           │

│ 4. OSSIGENO ad alto flusso              │
│                                         │

│                                         │

```

### Somministrazione Adrenalina

**Indicazioni:**

- Sintomi respiratori gravi (dispnea, stridore)
- Ipotensione/shock
- Sintomi cutanei + respiratori/cardiovascolari

**Dosaggio:**

| Paziente | Dose | Concentrazione |
|----------|------|----------------|
| Adulto > 30 kg | 0.5 mg | 1:1000 (1 mg/ml) |
| Bambino 15-30 kg | 0.3 mg | 1:1000 |
| Bambino < 15 kg | 0.15 mg | 1:1000 |

**Sede:** Faccia antero-laterale della coscia, IM

## Monitoraggio Post-Reazione

### Reazione Lieve/Moderata Risolta

- Osservazione in studio per 1-2 ore
- Non dimettere se sintomi persistono
- Prescrivere antistaminico per 3-5 giorni
- Consigliare visita allergologica

### Dopo Anafilassi

- Trasferimento SEMPRE in ospedale
- Possibile reazione bifasica (ritorno sintomi dopo 4-12 ore)
- Follow-up allergologico obbligatorio
