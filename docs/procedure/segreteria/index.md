# Segreteria

La segreteria è il cuore organizzativo dello studio e il primo punto di contatto con i pazienti. Un'efficiente gestione delle attività di front-office è fondamentale per la soddisfazione dei pazienti e l'ottimizzazione del lavoro clinico.

## Attività Principali

| Attività | Frequenza | Priorità |
|----------|-----------|----------|
| Risposta telefonica | Continua | Alta |
| Gestione agenda | Continua | Alta |
| Accoglienza pazienti | Continua | Alta |
| Conferme appuntamenti | Giornaliera | Alta |
| Gestione disdette | Al bisogno | Alta |
| Archiviazione documenti | Giornaliera | Media |
| Reportistica | Settimanale | Media |

### Copertura Minima

- Sempre almeno 1 persona in reception durante l'orario di apertura
- Telefono sempre presidiato
- Backup in caso di assenza

## Strumenti

### Software Gestionale

- Gestione agenda
- Anagrafica pazienti
- Cartelle cliniche
- Fatturazione
- Reportistica

### Comunicazione

- Telefono fisso + cordless
- Sistema SMS automatici
- Email studio
- WhatsApp Business (se attivo)

## KPI di Riferimento

| Indicatore | Obiettivo |
|------------|-----------|
| Tempo risposta telefonica | < 3 squilli |
| Tasso conferma appuntamenti | > 95% |
| Tasso no-show | < 5% |
| Soddisfazione pazienti | > 90% |
