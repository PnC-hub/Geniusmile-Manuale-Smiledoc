# Assistente alla Poltrona (ASO)

## Descrizione del Ruolo

L'Assistente di Studio Odontoiatrico (ASO) supporta l'odontoiatra durante le procedure cliniche e garantisce la corretta gestione dell'ambiente operativo.

## Qualifica Richiesta

- Attestato di qualifica ASO (corso regionale 700 ore)
- Oppure: esperienza pregressa + corso di aggiornamento

## Responsabilità Principali

### Assistenza Operativa

- Assistere l'odontoiatra durante le procedure
- Preparare il campo operatorio
- Passare strumenti e materiali
- Aspirare durante le procedure
- Gestire l'illuminazione

### Sterilizzazione e Igiene

- Eseguire il ciclo completo di sterilizzazione
- Gestire la tracciabilità degli strumenti
- Mantenere pulite le aree operative
- Gestire i rifiuti speciali
- Verificare funzionamento autoclave

### Gestione Materiali

- Controllare scorte della poltrona
- Segnalare necessità di riordino
- Verificare scadenze materiali
- Preparare i kit procedurali
- Gestire il magazzino di competenza

### Supporto al Paziente

- Accogliere il paziente in area clinica
- Preparare il paziente per la procedura
- Fornire istruzioni post-operatorie
- Gestire l'ansia del paziente
- Accompagnare il paziente a fine seduta

## Attività Quotidiane

### Prima dei Pazienti (8:30 - 9:00)

- [ ] Accendere apparecchiature
- [ ] Verificare livelli (acqua autoclave, aspirazione)
- [ ] Controllare sterilizzato disponibile
- [ ] Preparare prima poltrona
- [ ] Verificare agenda e procedure previste

### Durante le Procedure

- [ ] Preparare poltrona per ogni paziente
- [ ] Assistere durante procedura
- [ ] Gestire strumentario
- [ ] Riordinare tra un paziente e l'altro
- [ ] Avviare cicli di sterilizzazione

### Fine Giornata (dopo ultimo paziente)

- [ ] Sterilizzare ultimo strumentario
- [ ] Pulire e disinfettare superfici
- [ ] Svuotare separatori amalgama
- [ ] Verificare scorte per domani
- [ ] Spegnere apparecchiature
- [ ] Gestire rifiuti speciali

## Procedure per Tipo di Intervento

### Conservativa

| Fase | Attività ASO |
|------|--------------|
| Preparazione | Kit conservativa, diga, materiale |
| Durante | Aspirazione, passaggio strumenti |
| Dopo | Pulizia, sterilizzazione, registrazione |

### Chirurgia

| Fase | Attività ASO |
|------|--------------|
| Preparazione | Kit chirurgico sterile, farmaci |
| Durante | Aspirazione, divaricazione, passaggio |
| Dopo | Conta garze, sterilizzazione, smaltimento |

### Implantologia

| Fase | Attività ASO |
|------|--------------|
| Preparazione | Kit implantare, documentazione, materiali |
| Durante | Assistenza sterile, irrigazione, passaggio |
| Dopo | Verifica componenti, documentazione, sterilizzazione |

## Errori da Evitare

| Errore | Conseguenza | Prevenzione |
|--------|-------------|-------------|
| Strumento non sterile | Infezione | Verificare sempre indicatori |
| Materiale scaduto | Inefficacia trattamento | Controllo scadenze |
| Campo contaminato | Ripetere procedura | Attenzione ai movimenti |
| Mancata aspirazione | Disagio paziente | Concentrazione costante |
| Strumento sbagliato | Ritardo procedura | Conoscere le procedure |

## Gestione DPI

### DPI Obbligatori

| DPI | Quando |
|-----|--------|
| Guanti monouso | Sempre |
| Mascherina FFP2/chirurgica | Sempre |
| Occhiali/Visiera | Durante procedure |
| Divisa | Sempre |
| Scarpe chiuse | Sempre |

### Cambio DPI

- Guanti: tra un paziente e l'altro, se danneggiati
- Mascherina: ogni 4 ore o se umida/contaminata
- Divisa: giornaliera o se contaminata
