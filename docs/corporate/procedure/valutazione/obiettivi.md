# Obiettivi e Performance

## Sistema di Valutazione

Lo studio adotta un sistema di gestione per obiettivi (MBO) per allineare le performance individuali agli obiettivi dello studio.

## Ciclo di Valutazione

| Fase | Periodo | Attività |
|------|---------|----------|
| Definizione obiettivi | Gennaio | Assegnazione obiettivi annuali |
| Verifica intermedia | Giugno | Revisione e aggiustamenti |
| Valutazione finale | Dicembre | Consuntivo e feedback |
| Pianificazione | Dicembre-Gennaio | Obiettivi anno successivo |

## Tipologie di Obiettivi

### Obiettivi Quantitativi

Misurabili numericamente, legati a KPI specifici.

| Esempio per Ruolo | Obiettivo | Target |
|-------------------|-----------|--------|
| Segreteria | Tasso conferme appuntamenti | ≥ 95% |
| Segreteria | Riduzione no-show | < 5% |
| Igienista | Pazienti/giorno | ≥ 8 |
| ASO | Tempo turnover poltrona | < 10 min |
| Clinico | Completamento piani cura | ≥ 80% |

### Obiettivi Qualitativi

Legati a competenze, comportamenti, miglioramento.

| Area | Esempio |
|------|---------|
| Competenze tecniche | Acquisire certificazione X |
| Soft skills | Migliorare gestione conflitti |
| Innovazione | Proporre miglioramento processo |
| Team | Supportare onboarding nuovo collega |

### Obiettivi di Sviluppo

Legati alla crescita professionale.

| Esempio | Indicatore |
|---------|------------|
| Completare corso ECM su [tema] | Attestato |
| Acquisire competenza su [procedura] | Autonomia operativa |
| Assumere responsabilità [area] | Esito positivo |

## Obiettivi di Team

Oltre agli obiettivi individuali, possono essere definiti obiettivi di team.

### Esempio Obiettivi Team

| Obiettivo | Target | Peso |
|-----------|--------|------|
| Soddisfazione pazienti | NPS ≥ 50 | 30% |
| Riduzione reclami | < 2/mese | 20% |
| Puntualità agenda | ≥ 90% | 25% |
| Fatturato | ≥ obiettivo | 25% |

## Documentazione

### Archiviazione

- Schede obiettivi firmate
- Verbali verifiche intermedie
- Valutazioni finali
- Conservazione: 5 anni

### Riservatezza

Le valutazioni sono documenti riservati:
- Accessibili solo a interessato e responsabile
- Conservazione in luogo sicuro
- Non divulgare a colleghi
