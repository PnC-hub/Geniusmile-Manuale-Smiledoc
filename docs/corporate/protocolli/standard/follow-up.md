# Follow-up Post Trattamento

## Obiettivo

Garantire il benessere del paziente dopo i trattamenti e monitorare gli esiti clinici.

## Tipologie di Follow-up

### Follow-up Clinico Obbligatorio

| Trattamento | Tempistica | Modalità |
|-------------|------------|----------|
| Estrazione semplice | 7 giorni | Telefonico |
| Estrazione complessa/chirurgica | 7 giorni + visita controllo | Telefonico + Visita |
| Implanto | 7 gg + 1 mese + 3 mesi | Visita |
| Endodonzia | 7 giorni + 6-12 mesi RX | Telefonico + Visita |
| Chirurgia parodontale | 7-14 giorni | Visita |
| Protesi fissa cementata | 7-14 giorni | Telefonico/Visita |
| Sbiancamento | 48 ore | Telefonico |

### Follow-up di Cortesia

| Occasione | Tempistica | Modalità |
|-----------|------------|----------|
| Prima visita | 24-48 ore | Email/SMS |
| Primo trattamento | 24 ore | Telefonico |
| Trattamento invasivo | 24 ore | Telefonico |
| Completamento piano cura | 7 giorni | Telefonico |

## Procedure di Follow-up

### Telefonico Post-Chirurgico

**Chi:** Segreteria o ASO
**Quando:** 24-48 ore dopo l'intervento

**Script:**
> "Buongiorno Signor/a [Nome], sono [Nome] dello {{ clinic.full_name }}. La chiamo per sapere come sta dopo l'intervento di [tipo]. Come si sente? Ha avuto dolore? Gonfiore? Ha potuto seguire le indicazioni che le abbiamo dato?"

**Checklist domande:**

- [ ] Dolore (scala 1-10)
- [ ] Gonfiore
- [ ] Sanguinamento
- [ ] Febbre
- [ ] Assunzione farmaci
- [ ] Alimentazione
- [ ] Dubbi/domande

**Registrazione:**

| Data | Paziente | Intervento | Esito chiamata | Operatore |
|------|----------|------------|----------------|-----------|
| | | | ☐ OK ☐ Problema | |

### Gestione Problemi

| Problema Riferito | Azione |
|-------------------|--------|
| Dolore lieve-moderato | Rassicurare, verificare terapia |
| Dolore forte/persistente | Appuntamento urgente |
| Gonfiore normale | Rassicurare, ghiaccio |
| Gonfiore importante | Valutazione clinica |
| Sanguinamento lieve | Istruzioni compressione |
| Sanguinamento persistente | Appuntamento urgente |
| Febbre | Valutazione clinica |

### Follow-up Email/SMS

**Template conferma prima visita:**
> Gentile [Nome], grazie per aver scelto il nostro studio per la sua salute orale. Speriamo che la sua esperienza sia stata positiva. Per qualsiasi domanda siamo a disposizione al [telefono]. A presto!

**Template post-trattamento:**
> Gentile [Nome], volevamo assicurarci che stesse bene dopo il trattamento di [tipo]. Se ha dubbi o necessità non esiti a contattarci. Buona giornata!

## Gestione Richiami

### Processo

```
1. Generazione lista richiami (settimanale)
         ↓
2. Primo contatto (telefono/SMS)
         ↓
3. Se non risponde: secondo tentativo (3 giorni)
         ↓
4. Se non risponde: email
         ↓
5. Documentare esito in cartella
```

### Script Richiamo Igiene

> "Buongiorno Signor/a [Nome], sono [Nome] dello {{ clinic.full_name }}. La contatto perché sono passati [X mesi] dalla sua ultima seduta di igiene. È importante per la salute dei suoi denti mantenere controlli regolari. Quando preferisce fissare il prossimo appuntamento?"

## Documentazione

### Registrazione in Cartella

Ogni follow-up deve essere documentato:
- Data e ora
- Modalità (telefono, email, visita)
- Esito
- Eventuali problemi riferiti
- Azioni intraprese
- Operatore

### Conservazione

- Follow-up clinici: in cartella paziente
- Liste richiami: archivio gestionale
- Report: archivio qualità
