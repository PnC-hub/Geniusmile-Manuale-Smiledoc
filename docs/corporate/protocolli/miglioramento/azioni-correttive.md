# Azioni Correttive e Preventive

## Definizioni

| Termine | Significato |
|---------|-------------|
| **Non Conformità (NC)** | Mancato soddisfacimento di un requisito |
| **Azione Correttiva** | Azione per eliminare la causa di una NC |
| **Azione Preventiva** | Azione per prevenire il verificarsi di NC |
| **Correzione** | Azione per eliminare una NC (tratta l'effetto, non la causa) |

## Fonti di Non Conformità

| Fonte | Esempi |
|-------|--------|
| Audit interni | Risultati verifiche |
| Reclami pazienti | Lamentele formali |
| Segnalazioni personale | Problemi rilevati |
| Eventi avversi | Incidenti, near miss |
| Controlli di processo | KPI fuori target |
| Fornitori | Prodotti non conformi |

## Processo di Gestione

### Flusso Completo

```
1. Identificazione NC
         ↓
2. Registrazione
         ↓
3. Correzione immediata (se necessaria)
         ↓
4. Analisi causa radice
         ↓
5. Definizione azione correttiva
         ↓
6. Implementazione
         ↓
7. Verifica efficacia
         ↓
8. Chiusura
```

## Indicatori di Performance

| KPI | Target | Formula |
|-----|--------|---------|
| Tempo medio chiusura AC | < 30 giorni | Σ giorni / N. AC |
| AC chiuse nei tempi | ≥ 90% | Chiuse in tempo / Totale |
| Efficacia AC | ≥ 95% | AC efficaci / Totale verificate |
| Recidive | 0 | NC ripetute dopo AC |
