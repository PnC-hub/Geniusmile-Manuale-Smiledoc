# Gestione Reclami

## Obiettivo
Gestire in modo professionale ed efficace i reclami dei pazienti, trasformando le criticità in opportunità di miglioramento e fidelizzazione.

## Principi Fondamentali

!!! info "Il Reclamo è un'Opportunità"
    Un paziente che reclama è un paziente che vuole continuare il rapporto. Chi non reclama, semplicemente se ne va.

### Approccio

1. **Ascolto** attivo e rispettoso
2. **Empatia** con il paziente
3. **Tempestività** nella risposta
4. **Soluzione** concreta
5. **Apprendimento** per migliorare

| Canale | Gestione |

## Classificazione Reclami

### Per Gravità

| Livello | Descrizione | Gestione |
|---------|-------------|----------|
| Lieve | Disagio minore | Segreteria |
| Moderato | Insoddisfazione significativa | Coordinatore |
| Grave | Danno/Rischio | Direzione |
| Critico | Potenziale contenzioso | Direzione + Legale |

### Per Tipologia

| Categoria | Esempi |
|-----------|--------|
| Organizzativo | Attese, appuntamenti, orari |
| Relazionale | Comunicazione, cortesia |
| Clinico | Esito trattamento, dolore |
| Economico | Prezzi, preventivi, pagamenti |
| Strutturale | Comfort, pulizia, accessibilità |

## Procedura di Gestione

### Fase 1: Ricezione e Ascolto

**Comportamento:**

- Ascoltare senza interrompere
- Mantenere calma e professionalità
- Non giustificarsi immediatamente
- Non dare la colpa ad altri
- Prendere appunti

**Script iniziale:**

> "Mi dispiace per questa situazione. La prego, mi racconti cosa è successo. Vogliamo capire e trovare una soluzione."

### Fase 2: Raccolta Informazioni

Documentare:

| Campo | Informazione |
|-------|--------------|
| Data/ora reclamo | Quando è stato presentato |
| Paziente | Dati identificativi |
| Canale | Come è arrivato |
| Descrizione | Cosa è successo (parole del paziente) |
| Aspettativa | Cosa si aspetta il paziente |
| Documenti | Allegati, cartella clinica |

### Fase 3: Verifica

1. Consultare la cartella clinica
2. Parlare con il personale coinvolto
3. Verificare i fatti
4. Valutare la fondatezza

### Fase 4: Valutazione e Decisione

| Reclamo | Azione |
|---------|--------|
| Fondato | Riconoscere, scusarsi, risolvere |
| Parzialmente fondato | Spiegare, proporre compromesso |
| Infondato | Spiegare con rispetto |

### Fase 5: Risposta al Paziente

**Tempistiche:**

| Gravità | Risposta Entro |
|---------|----------------|
| Lieve | 24-48 ore |
| Moderato | 48-72 ore |
| Grave | 24 ore |
| Critico | Immediata |

**Contenuto risposta:**

1. Ringraziare per la segnalazione
2. Riassumere il problema (mostra comprensione)
3. Spiegare cosa è stato verificato
4. Comunicare la decisione
5. Proporre la soluzione
6. Scusarsi se appropriato
7. Assicurare impegno al miglioramento

### Fase 6: Soluzione

| Tipo | Esempi |
|------|--------|
| Rimborso | Parziale o totale |
| Ritrattamento | A carico dello studio |
| Sconto | Su future prestazioni |
| Scuse formali | Lettera |
| Altro | Da valutare caso per caso |

### Fase 7: Follow-up

- Verificare soddisfazione dopo la soluzione
- Documentare la chiusura
- Analizzare per prevenzione futura

## Reclami Clinici

### Particolare Attenzione

I reclami relativi a esiti clinici richiedono:

1. Coinvolgimento immediato dell'operatore
2. Valutazione clinica del caso
3. Eventuale consulto con colleghi
4. Documentazione accurata
5. Coinvolgimento assicurazione se necessario

### Cosa NON Fare

- Non ammettere responsabilità specifiche
- Non promettere rimborsi senza verifica
- Non alterare documentazione clinica
- Non ignorare il problema

### Coinvolgimento Assicurazione

Se il reclamo potrebbe configurare una responsabilità professionale:

1. Non assumere impegni
2. Contattare assicurazione RC
3. Seguire le indicazioni della compagnia
4. Coinvolgere legale se necessario

### Azioni di Miglioramento

Per ogni tema ricorrente:

1. Identificare la causa radice
2. Definire azione correttiva
3. Implementare
4. Verificare efficacia
5. Standardizzare se efficace
