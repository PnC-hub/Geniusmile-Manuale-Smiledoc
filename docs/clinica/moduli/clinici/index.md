## Principi di Documentazione

- **Accuratezza**: registrare fatti, non interpretazioni
- **Completezza**: tutti i dati rilevanti
- **Tempestività**: registrare subito dopo la seduta
- **Leggibilità**: scrittura chiara o digitale
- **Tracciabilità**: data, firma, identificazione
