# Diario Clinico

## Definizione

Il diario clinico è la registrazione cronologica di tutte le sedute e degli interventi effettuati sul paziente.

## Quando Compilare

- Dopo OGNI seduta clinica
- Immediatamente o appena possibile
- Prima che il paziente lasci lo studio (idealmente)

## Contenuto di Ogni Annotazione

### Dati Essenziali

| Campo | Contenuto |
|-------|-----------|
| Data | gg/mm/aaaa |
| Operatore | Nome/sigla identificativa |
| Assistente | Nome (se presente) |
| Elemento/i | Trattato/i |
| Procedura | Eseguita |

### Descrizione del Trattamento

**Cosa registrare:**

- Anestesia (tipo, quantità, sede)
- Procedura eseguita (in dettaglio)
- Materiali utilizzati (tipo, marca, lotto se rilevante)
- Eventuali complicanze intraoperatorie
- Esito della seduta
- Indicazioni post-operatorie date

### Aspetti da Documentare Sempre

- Farmaci somministrati o prescritti
- Reazioni avverse
- Rifiuto di trattamenti proposti
- Mancata collaborazione del paziente
- Comunicazioni importanti al paziente

## Esempio di Annotazione

```
DATA: 24/01/2026
OPERATORE: Dr. Rossi
ASSISTENTE: Sig.ra Verdi

ELEMENTO: 3.6

PROCEDURA:
Anestesia: Articaina 4% c/a 1:100.000, 1.8 ml, blocco n. alv. inf.
Isolamento con diga
Rimozione otturazione in amalgama
Cura canalare: strumentazione con Protaper Gold fino F2
Irrigazione con NaOCl 5% e EDTA 17%
Medicazione con Ca(OH)2
Otturazione provvisoria con Cavit

Prossima seduta: completamento endo

INDICAZIONI: Possibile lieve dolenzia, paracetamolo 1g se necessario

FIRMA: [Firma operatore]
```

## Errori da Evitare

| Errore | Perché Evitarlo |
|--------|-----------------|
| Annotazioni vaghe | Non documentano realmente l'attività |
| Compilazione tardiva | Rischio di dimenticare dettagli |
| Assenza di firma/data | Documento non valido |
| Cancellature | Sospetto di alterazione |
| Abbreviazioni non standard | Incomprensibili ad altri |

## Correzioni

Se si commette un errore:

1. **NON cancellare** con bianchetto o rendendo illeggibile
2. Barrare con una linea singola
3. Scrivere la correzione accanto
4. Siglare e datare la correzione

## Valore Legale

Il diario clinico ha valore di prova in caso di contenziosi. Una documentazione accurata è la migliore tutela per operatore e paziente.

## Conservazione

- Durata: illimitata
- Integrata nella cartella clinica
- Accessibile solo al personale autorizzato
