# Requisiti Strutturali

## Requisiti Minimi dei Locali

### Sala Operativa (Riunito)

- Superficie minima: secondo normativa regionale (es. 9-12 mq)
- Illuminazione naturale e artificiale adeguata
- Ventilazione adeguata
- Pavimenti e pareti lavabili
- Lavabo con comando non manuale

### Sala d'Attesa

- Dimensioni adeguate al numero di riuniti
- Areazione
- Servizi igienici per i pazienti

### Sterilizzazione

- Area dedicata (o zona distinta)
- Percorso sporco/pulito
- Autoclave conforme

### Servizi Igienici

- Almeno uno per i pazienti
- Almeno uno per il personale (se >5 addetti)
- Accessibilità disabili (se richiesta)

## Impianti

| Impianto | Requisito |
|----------|-----------|
| Elettrico | Dichiarazione di conformità |
| Idrico | Acqua potabile |
| Scarichi | Conformi alle norme |
| Climatizzazione | Adeguata |
| Gas medicali | Se presenti, conformi |

## Accessibilità

Secondo normativa vigente sull'abbattimento barriere architettoniche.
