# Sincope Vasovagale

## Obiettivo
Gestire correttamente la sincope vasovagale, l'emergenza più frequente in ambito odontoiatrico.

## Definizione

La **sincope vasovagale** è una perdita di coscienza transitoria causata da una riduzione temporanea del flusso sanguigno cerebrale, dovuta a stimolazione del nervo vago.

## Fattori Predisponenti

### Legati al Paziente

- Ansia e paura
- Digiuno prolungato
- Disidratazione
- Caldo eccessivo
- Posizione seduta prolungata
- Storia di precedenti sincopi

### Legati alla Procedura

- Vista del sangue o degli strumenti
- Iniezione di anestetico
- Dolore
- Stimoli spiacevoli

## Fasi della Sincope

### Fase Prodromica (Pre-sincope)

!!! warning "Riconoscere i Segnali"
    Identificare questa fase permette di prevenire la perdita di coscienza completa.

**Sintomi da riconoscere:**

- Pallore improvviso
- Sudorazione fredda
- Nausea
- Sensazione di calore
- Visione offuscata
- Ronzio alle orecchie
- Debolezza
- "Mi sento strano/male"

### Fase Sincopale

- Perdita di coscienza
- Ipotonia muscolare
- Polso debole e bradicardico
- Pressione arteriosa bassa
- Possibili movimenti clonici brevi

### Fase di Recupero

- Ripresa coscienza (generalmente rapida)
- Confusione transitoria
- Debolezza persistente
- Possibile nausea

## Protocollo di Gestione

### Se Sintomi Prodromici

```
PREVENZIONE SINCOPE

1. INTERROMPERE immediatamente il trattamento

2. POSIZIONE TRENDELENBURG
   - Abbassare lo schienale
   - Alzare le gambe
   - Slacciare indumenti stretti

3. RASSICURARE con voce calma

4. STIMOLARE
   - Impacco freddo su fronte/nuca
   - Far respirare profondamente

5. MONITORARE
   - Osservare colorito
   - Verificare se migliora

6. NON far alzare fino a completo recupero
```

### Se Sincope Completa

```
┌─────────────────────────────────────────┐
│        GESTIONE SINCOPE COMPLETA        │
├─────────────────────────────────────────┤
│ 1. Posizione supina, gambe alzate       │
│    (30-40 cm)                           │
│                                         │
│ 2. Liberare le vie aeree                │
│    - Estendere il capo                  │
│    - Sublussare la mandibola            │
│                                         │
│ 3. Verificare il respiro (GAS 10 sec)   │
│                                         │
│ 4. Se respira → attendere ripresa       │
│    Se NON respira → RCP + 118           │
│                                         │
│ 5. Allentare indumenti stretti          │
│                                         │
│ 6. Applicare impacco freddo             │
│                                         │
│ 7. Monitorare fino a ripresa            │
└─────────────────────────────────────────┘
```

### Dopo la Ripresa

1. **Non far alzare** immediatamente
2. Attendere almeno **10-15 minuti** in posizione supina
3. Far alzare **gradualmente**:
   - Prima seduto
   - Poi in piedi con supporto
4. Offrire acqua zuccherata
5. Verificare stabilità prima di dimettere

## Diagnosi Differenziale

!!! danger "Attenzione"
    Non tutte le perdite di coscienza sono sincopi vasovagali.

| Condizione | Elementi distintivi |
|------------|---------------------|
| Sincope vasovagale | Prodromi tipici, recupero rapido e completo |
| Ipoglicemia | Diabetico, digiuno, tremore, sudorazione |
| Crisi epilettica | Movimenti tonici-clonici, morsicatura lingua |
| Aritmia cardiaca | Assenza prodromi, cardiopatico |
| Ictus | Deficit neurologici focali, asimmetria |

### Quando Chiamare il 118

- Perdita di coscienza > 1-2 minuti
- Mancato recupero completo
- Sintomi neurologici
- Dolore toracico
- Convulsioni
- Paziente cardiopatico
- Dubbi sulla natura dell'evento

## Prevenzione

### Identificazione Pazienti a Rischio

In anamnesi chiedere:

- "Le è mai capitato di svenire?"
- "Ha paura degli aghi o del sangue?"
- "Si sente ansioso/a prima delle visite dal dentista?"

### Strategie Preventive

| Strategia | Applicazione |
|-----------|--------------|
| Posizione | Poltrona leggermente reclinata |
| Comunicazione | Avvisare prima di ogni azione |
| Distrazione | Conversare, musica |
| Tecnica atraumatica | Anestetico topico, iniezione lenta |
| Timing | Non a digiuno, non dopo sforzi |
| Ambiente | Temperatura confortevole, areazione |

### Per Pazienti Altamente Ansiosi

Considerare:

- Sedute brevi
- Sedazione cosciente
- Premedicazione ansiolitica (se indicata)
- Tecniche di rilassamento

## Documentazione

Registrare in cartella clinica:

- [ ] Data e ora dell'evento
- [ ] Circostanza (cosa si stava facendo)
- [ ] Sintomi riferiti/osservati
- [ ] Durata perdita di coscienza
- [ ] Trattamento effettuato
- [ ] Tempo di recupero
- [ ] Condizioni alla dimissione
- [ ] Raccomandazioni date

## Follow-up

### Indicazioni al Paziente

- Non guidare per 24 ore se sincope completa
- Mangiare prima di uscire
- Farsi accompagnare
- Contattare se sintomi persistono

### Visite Successive

- Segnalare in cartella "RISCHIO SINCOPE"
- Applicare protocollo preventivo
- Comunicare con il paziente

## Responsabilità

| Ruolo | Compito |
|-------|---------|
| Operatore | Riconoscimento, gestione, decisioni |
| Assistente | Posizionamento, monitoraggio, supporto |
| Segreteria | Gestione altri pazienti, eventuale 118 |
