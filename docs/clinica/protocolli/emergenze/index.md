# Emergenze Mediche

La gestione delle emergenze mediche in studio odontoiatrico richiede preparazione, protocolli chiari e addestramento regolare di tutto il personale.

## Principi Generali

!!! danger "Numeri di Emergenza"
    **118** - Emergenza Sanitaria

    **Indirizzo Studio:** {{ clinic.full_address }}

### Catena di Sopravvivenza

1. **Riconoscimento precoce** dell'emergenza
2. **Attivazione** dei soccorsi (118)
3. **Supporto** delle funzioni vitali (BLS)
4. **Defibrillazione** precoce (se disponibile DAE)
5. **Trattamento** avanzato (personale 118)

## Emergenze più Comuni

| Emergenza | Frequenza | Gravità |
|-----------|-----------|---------|
| Sincope vasovagale | Frequente | Bassa |
| Ipotensione ortostatica | Frequente | Bassa |
| Crisi ipoglicemica | Occasionale | Media |
| Reazione allergica | Rara | Alta |
| Crisi ipertensiva | Rara | Media-Alta |
| Crisi anginosa | Rara | Alta |
| Arresto cardiaco | Molto rara | Critica |

## Ruoli in Emergenza

```
SCHEMA RUOLI

OPERATORE (Leader)
├── Valuta il paziente
├── Coordina le azioni
├── Esegue manovre salvavita
└── Interfaccia con 118

ASSISTENTE 1
├── Chiama il 118
├── Porta kit emergenza
├── Assiste l'operatore
└── Documenta tempi/azioni

ASSISTENTE 2 / SEGRETERIA
├── Libera le vie di accesso
├── Allontana altri pazienti
├── Accoglie il 118
└── Recupera documentazione paziente
```

## Prevenzione

### Anamnesi Accurata

Identificare pazienti a rischio:

- Cardiopatici
- Diabetici
- Allergici
- Ipertesi
- Ansiosi/fobici

### Monitoraggio

Per pazienti a rischio:

- Rilevazione PA pre-trattamento
- Monitoraggio durante procedure lunghe
- Attenzione a sintomi prodromici

## Formazione Obbligatoria

| Corso | Frequenza | Personale |
|-------|-----------|-----------|
| BLS-D | Ogni 2 anni | Tutto il personale |
| Gestione emergenze | Annuale | Tutto il personale |
| Simulazioni | Semestrale | Tutto il personale |

## Documentazione

Ogni emergenza deve essere documentata:

- Data, ora, luogo
- Dati del paziente
- Descrizione dell'evento
- Azioni intraprese
- Esito
- Firma degli operatori

## Responsabilità

| Ruolo | Compito |
|-------|---------|
| Direttore Sanitario | Protocolli, formazione, kit |
| Tutto il personale | Conoscenza protocolli, addestramento |
| Coordinatore | Verifiche periodiche, simulazioni |
