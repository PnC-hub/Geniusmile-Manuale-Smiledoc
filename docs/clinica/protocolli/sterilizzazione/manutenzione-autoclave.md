# Manutenzione Autoclave

## Obiettivo
Garantire il corretto funzionamento dell'autoclave attraverso una manutenzione programmata e documentata.

## Manutenzione Giornaliera

### Prima dell'Utilizzo

!!! tip "Checklist Mattutina"
    Da eseguire ogni giorno prima del primo ciclo operativo.

- [ ] Verificare livello acqua demineralizzata
- [ ] Controllare assenza di residui nella camera
- [ ] Verificare guarnizione porta (integrità, pulizia)
- [ ] Eseguire ciclo di riscaldamento a vuoto
- [ ] Eseguire **Test Bowie-Dick** (primo ciclo del giorno)

### Test Bowie-Dick

**Scopo:** Verificare la penetrazione del vapore e l'evacuazione dell'aria.

**Procedura:**

1. Camera vuota e fredda
2. Posizionare il test pack al centro del ripiano inferiore
3. Avviare ciclo dedicato (134°C - 3,5 min)
4. Verificare viraggio uniforme dell'indicatore
5. Registrare esito sul registro

**Interpretazione:**

| Risultato | Azione |
|-----------|--------|
| Viraggio uniforme | OK - Procedere con cicli operativi |
| Viraggio non uniforme | NON OK - Non utilizzare, chiamare assistenza |
| Nessun viraggio | NON OK - Verificare ciclo, ripetere test |

### A Fine Giornata

- [ ] Scaricare acqua residua (se previsto dal modello)
- [ ] Pulire camera con panno umido
- [ ] Lasciare porta socchiusa
- [ ] Pulire esterno macchina

## Manutenzione Settimanale

### Controlli da Effettuare

| Controllo | Procedura |
|-----------|-----------|
| Filtro aria | Verificare pulizia, sostituire se necessario |
| Scarico | Verificare assenza ostruzioni |
| Serbatoio acqua | Pulire e disincrostare |
| Guarnizioni | Controllare usura e integrità |

### Test Biologico (Spore Test)

!!! warning "Importante"
    Il test biologico è l'unico metodo per verificare l'effettiva capacità sterilizzante.

**Frequenza:** Settimanale (consigliato) o almeno mensile (obbligatorio)

**Procedura:**

1. Posizionare fiala con spore (Geobacillus stearothermophilus) nel carico più impegnativo
2. Eseguire ciclo standard
3. Incubare la fiala secondo istruzioni (24-48h a 55-60°C)
4. Verificare assenza di crescita
5. Registrare esito

**Interpretazione:**

| Indicatore | Risultato | Azione |
|------------|-----------|--------|
| Giallo (negativo) | Sterilizzazione efficace | OK |
| Viola (positivo) | Crescita batterica | BLOCCARE autoclave |

### In Caso di Test Positivo

1. **SOSPENDERE** immediatamente l'uso
2. Richiamare tutto il materiale del lotto
3. Contattare assistenza tecnica
4. Documentare la non conformità
5. Non riutilizzare fino a risoluzione

## Manutenzione Programmata

### Mensile

- Pulizia approfondita camera
- Verifica calibrazione sensori
- Controllo tenuta porta
- Verifica sistema di sicurezza

### Trimestrale

- Sostituzione guarnizione porta (se necessario)
- Controllo valvole di sicurezza
- Verifica sistema idraulico
- Backup dati (se autoclave elettronica)

### Annuale (Manutenzione Ordinaria)

Da effettuarsi a cura di **tecnico autorizzato**:

- Calibrazione strumenti di misura
- Verifica conformità parametri
- Sostituzione parti soggette a usura
- Rilascio certificato di conformità

## Problemi Comuni e Soluzioni

| Problema | Possibile Causa | Soluzione |
|----------|-----------------|-----------|
| Buste bagnate | Sovraccarico, asciugatura insufficiente | Ridurre carico, aumentare asciugatura |
| Ciclo interrotto | Guarnizione usurata | Sostituire guarnizione |
| Allarme temperatura | Sensore guasto | Chiamare assistenza |
| Acqua insufficiente | Serbatoio vuoto | Rabboccare acqua demineralizzata |
| Indicatore non vira | Vapore non adeguato | Test Bowie-Dick, verificare funzionamento |
