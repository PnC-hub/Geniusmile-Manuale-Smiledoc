# Sterilizzazione

La sterilizzazione è un processo fondamentale per prevenire le infezioni crociate e garantire la sicurezza di pazienti e operatori.

## Principi Generali

Lo {{ clinic.full_name }} adotta il principio della **catena di sterilità**:

1. **Decontaminazione** - Rimozione dei residui organici
2. **Lavaggio** - Pulizia meccanica o manuale
3. **Risciacquo** - Eliminazione dei residui di detergente
4. **Asciugatura** - Preparazione per il confezionamento
5. **Confezionamento** - Imbustamento con indicatori
6. **Sterilizzazione** - Ciclo in autoclave
7. **Stoccaggio** - Conservazione in ambiente protetto
