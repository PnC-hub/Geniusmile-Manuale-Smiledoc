# Comunicazione e Marketing

La comunicazione efficace con i pazienti è fondamentale per costruire relazioni durature, fidelizzare e acquisire nuovi pazienti.

## Obiettivi

- Fidelizzare i pazienti esistenti
- Acquisire nuovi pazienti
- Mantenere una comunicazione costante
- Gestire la reputazione dello studio

## Canali di Comunicazione

| Canale | Utilizzo | Target |
|--------|----------|--------|
| Telefono | Personale, urgente | Tutti |
| SMS | Promemoria, recall | Tutti |
| Email | Informativo, documentale | Chi ha email |
| WhatsApp | Informale, rapido | Su consenso |
| Social | Awareness, engagement | Pubblico generale |
| Sito web | Informativo | Potenziali pazienti |

## Calendario Comunicazioni

### Comunicazioni Periodiche

| Tipo | Frequenza | Contenuto |
|------|-----------|-----------|
| Recall igiene | 6 mesi | Promemoria controllo |
| Recall controllo | 12 mesi | Promemoria visita |
| Auguri | Festività | Natale, Pasqua |
| Newsletter | Trimestrale | Novità, consigli |

### Piano Annuale

```
NOV: -

```

## Privacy e Consensi

!!! warning "GDPR"
    Tutte le comunicazioni devono rispettare la normativa sulla privacy.

### Consensi Necessari

| Tipo Comunicazione | Consenso |
|--------------------|----------|
| Promemoria appuntamenti | Base (legittimo interesse) |
| Recall sanitari | Base (legittimo interesse) |
| Comunicazioni marketing | Specifico (opt-in) |
| Newsletter | Specifico (opt-in) |

### Gestione Opt-out

- Rispettare sempre le richieste di cancellazione
- Aggiornare il database entro 48h
- Mantenere registro delle richieste

| Ruolo | Compito |
