# Gestione Fornitori

## Obiettivo
Costruire e mantenere rapporti efficaci con i fornitori per garantire qualità, convenienza e affidabilità delle forniture.

## Anagrafica Fornitori

### Dati da Registrare

| Campo | Informazione |
|-------|--------------|
| Ragione sociale | Nome completo |
| P.IVA / CF | Dati fiscali |
| Indirizzo | Sede legale |
| Telefono | Principale + urgenze |
| Email | Ordini + amministrazione |
| Referente | Nome commerciale |
| IBAN | Per bonifici |
| Condizioni | Pagamento, consegna |

### Categorie Fornitori

| Categoria | Esempi |
|-----------|--------|
| Materiale consumo | Depositi dentali |

| Laboratorio | Odontotecnici |
| Farmaci | Grossisti farmaceutici |
| Servizi | Manutenzione, software |
| Utenze | Energia, telefonia |

## Selezione Fornitori

### Criteri di Valutazione

| Criterio | Peso | Note |
|----------|------|------|
| Qualità prodotti | Alto | Certificazioni, recensioni |
| Prezzi | Alto | Confronto mercato |
| Affidabilità consegne | Alto | Puntualità, completezza |
| Condizioni pagamento | Medio | Dilazioni, sconti |
| Servizio clienti | Medio | Reattività, supporto |
| Ampiezza catalogo | Basso | Convenienza ordini unici |

### Processo di Qualifica

1. **Ricerca** fornitori alternativi
2. **Richiesta** preventivi/condizioni
3. **Valutazione** comparativa
4. **Test** con ordine di prova
5. **Valutazione** performance
6. **Decisione** inserimento in anagrafica

### Documenti da Richiedere

- Visura camerale
- Certificazioni qualità (ISO se applicabile)
- Listino prezzi
- Condizioni commerciali
- Referenze (opzionale)

## Condizioni Commerciali

### Negoziazione

**Elementi negoziabili:**

| Elemento | Obiettivo |
|----------|-----------|
| Sconto base | % su listino |
| Sconto quantità | Per ordini grandi |
| Pagamento | 30-60-90 giorni |
| Trasporto | Franco destino |
| Resi | Politica flessibile |
| Promozioni | Accesso anticipato |

### Accordi Quadro

Per fornitori principali, concordare:

- Condizioni per l'anno
- Volumi presunti
- Prezzi bloccati
- Servizi inclusi

## Gestione Ordini

### Ripartizione Acquisti

!!! tip "Diversificazione"
    Non dipendere da un unico fornitore per categorie critiche.

**Regola suggerita:**

- Fornitore principale: 60-70%
- Fornitore secondario: 20-30%
- Fornitori occasionali: 10%

### Ordine a Fornitore

1. Preparare lista prodotti
2. Verificare disponibilità (se necessario)
3. Inviare ordine scritto
4. Richiedere conferma
5. Archiviare documentazione

## Reclami e Resi

### Gestione Non Conformità

**Tipologie:**

| Problema | Azione |
|----------|--------|
| Prodotto danneggiato | Foto + reclamo + reso |
| Quantità errata | Segnalazione + rettifica |
| Prodotto errato | Reso + sostituzione |
| Scadenza breve | Valutazione + eventuale reso |
| Difetto qualità | Documentazione + reso |

### Procedura Reclamo

1. **Documentare** il problema (foto, descrizione)
2. **Contattare** il fornitore tempestivamente
3. **Richiedere** soluzione (sostituzione, nota credito)
4. **Tracciare** il reclamo
5. **Verificare** la risoluzione
6. **Aggiornare** valutazione fornitore
